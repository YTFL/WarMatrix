# **App Name**: TacticalNexus Console

## Core Features:

- Central Tactical Map: Visualize the battlefield with a grid-based terrain, displaying friendly unit markers, enemy activity, movement paths, and strategic objectives using distinct glowing symbols and lines.
- Left Intelligence Panel: Display real-time battlefield intelligence, including terrain data, weather conditions, sensor detections, communication status, and logistics updates, as a live monitoring feed.
- Right Strategic Analysis Panel: Provide AI-generated operational assessments, risk analyses, predicted enemy behaviors, and recommended next actions, resembling a dynamic military briefing tool.
- Bottom Command Console: Enable operators to issue commands through selectable options like 'Deploy Reconnaissance Units' or 'Launch Counteroffensive', with a prominent execution button.
- System Header Bar: Display crucial system-level information, such as the application title, simulation turn number, and overall system status (Active/Awaiting Command).
- Simulation Result Display: Present an operation update panel after commands are executed, detailing the command, updated success probability, operational risk, and immediate battlefield outcome.

## Style Guidelines:

- Base background color: Deep navy-black (hex: #0A0F1C) to simulate a dim command room environment.
- Secondary background panels: Slightly lighter dark navy or slate (hex: #111827 or #141A2E) for separate digital displays embedded into a workstation.
- Tactical battlefield map background: Dark blue-grey (hex: #0F172A or #0B1324) to ensure the map stands out visually.
- Grid lines on the map: Muted blue-grey (hex: #1F2A44) for subtle but visible grid overlays, creating a digital tactical map projection.
- Primary text color: Soft white (hex: #E5E7EB) for high readability against dark backgrounds without being overly bright.
- Secondary text color: Light grey (hex: #9CA3AF) for supplementary information, maintaining readability.
- Friendly forces display color: Glowing green (hex: #22C55E) to clearly identify allied units.
- Enemy forces display color: Alert red (hex: #EF4444) for immediate identification of threats.
- Objectives or warnings display color: Amber yellow (hex: #F59E0B) for critical locations and warning indicators.
- Primary system highlight color: Electric cyan (hex: #22D3EE) for charts, radar displays, and system indicators, appearing as glowing data lines.
- Secondary system highlight color: Neon blue (hex: #38BDF8) for additional cool-toned tactical data and indicators.
- Default button background: Dark slate (hex: #1F2937) for command console buttons.
- Button hover state: Slightly brighter slate (hex: #374151) for interactive feedback.
- Primary command button (execute): Deep teal or cyan highlight (hex: #0891B2) to emphasize critical actions.
- Status indicator for stable/success: Green (hex: #22C55E).
- Status indicator for warning: Yellow (hex: #F59E0B).
- Status indicator for critical/threat: Red (hex: #EF4444).
- Active panel highlight glow: Subtle cyan-blue (hex: #06B6D4) for selected or active panels, with minimal intensity to maintain professionalism.
- Headline font: 'Space Grotesk' (sans-serif) for its computerized, techy, scientific feel, suitable for system titles and key headers.
- Body text font: 'Inter' (sans-serif) for neutral, modern, and highly legible text across all data panels and console inputs.
- Utilize small, glowing geometric symbols or dots for unit markers and other map elements, reinforcing the digital screen aesthetic. Focus on clear, high-contrast silhouettes.
- The screen is structured into five distinct, simultaneously visible areas: a System Header Bar at the top, a Central Tactical Map as the focal point, Left Intelligence and Right Strategic Analysis vertical panels flanking the map, and a Bottom Command Console for user input.
- Implement subtle glowing and pulsing effects for active UI elements and accent indicators. Use smooth, quick transitions for information updates, command execution feedback, and state changes to convey a 'live' operational environment.
